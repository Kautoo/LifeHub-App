# 🎨 LifeHub Logo Generation Prompts

## 🎯 Main Prompt (For Gemini / ChatGPT)

### Version 1: Modern & Minimal
```
Design a modern, minimal app logo for "LifeHub" - an all-in-one life management productivity app.

ICON: A stylized hexagonal hub shape with interconnected nodes representing different life areas (work, health, finance, learning). The hub should feel connected and organized.

STYLE: 
- Modern flat design with subtle gradient
- Clean geometric shapes
- Minimalist but recognizable

COLORS:
- Primary gradient: Purple (#6C63FF) to Cyan (#00D2FF)
- Background: Transparent or Deep Dark (#0D1117)

REQUIREMENTS:
- Works as app icon (512x512px)
- Scalable from 48dp to 512dp
- Looks good on dark backgrounds
- Simple enough to be recognizable at small sizes
- No text in the icon (text added separately)
- Premium, modern feel
- Clean edges, no pixelation
```

## 💡 Tips for Best Results

### For Gemini:
- Use "Generate an image of..." prefix
- Be specific about dimensions
- Ask for multiple variations
- Request "app icon" or "logo" specifically

### For ChatGPT (DALL-E):
- Start with "Create a logo..."
- Specify "vector style" or "flat design"
- Ask for "transparent background"
- Request "high resolution, scalable"

### General Tips:
1. Generate 5-10 variations, pick the best
2. Ask for the logo on both light AND dark backgrounds
3. Request simplified versions for small sizes
4. Get a version without text (for app icon)
5. Ask for color palette extraction from the logo
