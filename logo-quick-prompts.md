## 🎨 Best Prompt (Recommended Start)

Use this one first — it's the most detailed and likely to give great results:

### For Gemini:
```
Generate a premium app icon for "LifeHub", a modern productivity app for students. 

Design: An abstract hub symbol made of 4 connected nodes arranged in a diamond/hexagonal pattern. The central node is slightly larger. Nodes are connected by thin curved lines. All elements are white with subtle glow effect.

Background: Smooth gradient from deep purple (#6C63FF) at top-left to bright cyan (#00D2FF) at bottom-right.

Style: Clean, geometric, modern flat design. Premium SaaS aesthetic. Recognizable at small sizes.

Format: 512x512 square, centered with padding. Show on dark background (#0D1117) to preview.
```

### For ChatGPT/DALL-E:
```
A premium app icon for "LifeHub" productivity app. Abstract network hub symbol: 4 white glowing dots connected by thin curved lines, arranged in a diamond pattern. Gradient background from purple (#6C63FF) to cyan (#00D2FF). Clean vector style, modern flat design, 512x512 pixels. Show on dark background.
```

## 📋 What to Do After Generating

1. Pick your favorite design
2. Generate variations (different node arrangements, different gradient directions)
3. Test at small size (make it 48x48 to see if still recognizable)
4. Get PNG with transparent background
5. We'll add it to the Android app as app icon!
